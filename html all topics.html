HTML Topics

HTML:
HTML stands for Hyper Text Markup Language. HTML is the standard markup language for creating Web pages. HTML describes the structure of a Web page. HTML consists of a series of elements. HTML elements tell the browser how to display the content.
Heading tags:
Header tags, also known as heading tags, are used to separate headings and subheadings on a webpage. They rank in order of importance, from H1 to H6, with H1s usually being the title. Header tags improve the readability and SEO of a webpage.
    -Heading tags:
  • h1-highest heading tag
  • h2-second heading tag
  • h3-third heading tag
  • h4-fourth heading tag
  • h5-fifth heading tag
  • h6-least heading tag
Peragraph tag
The aragraph ptag defines a paragraph. Browsers automatically add a single blank line before and after each element.
Formating tags:
  1. Bold and strong tag
  2. Italic and em tag
  3. underline tag
  4. subscription tag
  5. super tag
  6. del tag
  7. mark tag
  8. blockquote tag
  9. pre tag
  10. address tag
Block-Level element:
 
A block-level element always starts on a new line and takes up the full width available.
In-line elements:
 
An inline element does not start on a new line and it only takes up as much width as necessary.
 
Tables:
  • An HTML table consists of one table element and one or more tr, th, and td elements.
  • The 'tr' element defines a table row.
  • the 'th' element defines a table header
  • the td element defines a table cell
Img tag:
The tag is used to embed an image in an HTML page. Images are not technically inserted into a web page; images are linked to web pages. The tag creates a holding space for the referenced image.
  • src-the source location
  • alt-the alternate text
Anchor tag:
Anchors in HTML are used to create hyperlinks that allow users to navigate between different web pages or sections within the same page.
  1. href
  2. data
URL Structure:
A URL consists of ten parts:
  1. the schme
  2. subdomain
  3. second level domain
  4. subdirectory
  5. parameter
  6. port
  7. path
  8. query
  9. fragment
List tags:
In HTML, a list is an element used to create organized and structured lists of items.
  • order list
  • un-order list
  • description lists
